# SportParts
# Introduccion:
En este repositorio llevare a cabo la elaboracion de un programa para un proyecto que tengo pensado abordar en un futuro, tratara de un sistema de gestion de inventario para una tienda deportiva. El proyecto estara totalmente hecho en java ero tengo pensado estudiar en las vacaciones crecion de paginas web, y hacerlo multiplataformas, que se pueda usar tanto en celular mediante una app o en la pagina, como en computadores con la aplicacion de escritorio. Mis conocimientos hoy 12/11/2025 son totalmente basicos, apenas cuento con lo basico de java y tengo que dominar completamente la programacion orientada a objetos, pero eso se ira solucionando con el tiempo. Como reto importante para la elaboracion de este proyecto, tengo totalmente prohibido usar inteligencia artificial, ya sea para preguntar algo, o para generar el codigo completo, todo estara hecho con conocimientos que ire adquiriendo ya sea viendo videos en youtube, haciendo algun curso por ahi, o incluo, de libros, pero el uso de inteligencia artificial no sera recibido, esto es debido a que mas que generar de una vez el programa para mi proyecto a futuro, es un repaso y un complemento de todo lo que vi este semestre en programacion 2 en proceso de formacion como ingeniero de sistemas. A continuacion muestro las opciones permitidas y no permitidas para elaborar:
### 1. Opciones permitidas:
 * Videos de Youtube
 * Foros
 * Paginas de Conocimiento Publico
 * Codigo de otros creadores
### 2. Opciones no permitidas:
 - Uso de inteligencia artificial
   - Chat gpt
   - Copilot
   - **Claude**
   - Ni ninguna por el estilo
 - Ayuda externa de algun amigo o conocido
